# Contributing to Hospital Readmission Risk Predictor

We welcome contributions to improve data preprocessing, model calibration, or fairness evaluation.

- **Fairness Rules:** Changes to models must run through the fairness evaluation suite.
- **Code Standards:** Document all helper classes and maintain clean separation between preprocessors and model layers.
